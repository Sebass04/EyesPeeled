# EyesPeeled

EyesPeeled è un'avventura grafica Escape Room sviluppata interamente in Java come progetto per il corso di Metodi Avanzati di Programmazione.

## Descrizione

Il giocatore deve esplorare diverse stanze, raccogliere oggetti, risolvere enigmi e trovare la via d'uscita da una misteriosa Escape Room chiamata "EyesPeeled".

L'interazione avviene tramite una GUI che consente di muoversi tra le stanze, gestire l'inventario e utilizzare gli oggetti raccolti durante l'avventura.

## Avvio del progetto

1. Clonare il repository:
   git clone https://github.com/Sebass04/EyesPeeled.git

2. Aprire il progetto con NetBeans

3. Eseguire la classe principale:
   Serino.java

## Funzionalità

- Interfaccia grafica (GUI)
- Sistema di inventario
- Gestione degli oggetti di gioco
- Enigmi e puzzle progressivi
- Sistema di salvataggio/caricamento
- Database H2 per la gestione dei dati
- Colonna sonora integrata
- Persistenza dei dati tramite serializzazione Java

## Tecnologie utilizzate

- Java
- Java Swing
- H2 Database
- OOP (Object-Oriented Programming)
- Java Serialization
- Multithreading

## Struttura del progetto

- `AdvObject` → gestione degli oggetti di gioco
- `DataBaseInit` → inizializzazione e gestione database
- `Save` → salvataggio e caricamento della partita
- `Soundtrack` → gestione della musica
- `Serino` → avvio dell'applicazione

## Competenze sviluppate

- Progettazione software
- Programmazione orientata agli oggetti
- Gestione database relazionali
- Persistenza dei dati
- Sviluppo GUI
